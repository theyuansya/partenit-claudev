# Contributing to Claudev

Thanks for your interest in contributing!

## Getting started

1. Fork the repo and clone it
2. Copy `.env.example` to `.env` and fill in your values
3. Run locally: `docker compose up --build`
4. Make your changes and test them

## Pull requests

- Keep PRs focused — one feature or fix per PR
- Update README if you add new env variables or features
- Test with a real Jira project if possible

## Bug reports

Open an issue with:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Your environment (Railway / Docker / local)

## Ideas and feature requests

Open an issue! We're happy to discuss before you start coding.
